# Gym Management API

REST API developed with Java and Spring Boot for gym management studies.

## 🚀 Technologies

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Lombok

## 📚 Features

- Student registration
- Physical assessment registration
- Workout management
- Exercise management
- JPA relationships
- Global exception handling

## 🏗️ Architecture

```bash
controller
service
repository
model
dto
handler
exception
```

## 🔗 Relationships

- OneToOne
- OneToMany
- ManyToOne

## ▶️ Running the project

```bash
git clone https://github.com/SEU-USUARIO/gym-management-api.git
```

```bash
cd gym-management-api
```

```bash
./mvnw spring-boot:run
```
